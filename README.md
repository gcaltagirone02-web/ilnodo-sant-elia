# Il Nodo 2.0 — Beach Bar a Sant'Elia

Sito vetrina ad alte prestazioni per **Il Nodo Cocktail & Beach Bar** situato a Sant'Elia (PA). Il progetto è focalizzato sulla Local SEO, la conversione immediata e un'estetica premium "Solar".

## 🚀 Tecnologie Core

- **Framework:** [Astro 5](https://astro.build/) (SSG - Static Site Generation)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **CMS:** [Sanity.io](https://www.sanity.io/) (Headless CMS)
- **Hosting:** [Cloudflare Pages](https://pages.cloudflare.com/)
- **Performance:** Obiettivo Lighthouse 100/100 (Zero-JS by default)

## 🛠️ Struttura del Progetto

```text
/
├── _bmad/               # Configurazione e workflow BMAD Method
├── _bmad-output/        # Artifact di planning e implementazione
├── docs/                # Documentazione di supporto e inventory
├── public/              # Asset statici pubblici
└── src/
    ├── assets/          # Immagini ottimizzate via Astro
    ├── components/      # Componenti UI (Astro/Tailwind)
    ├── layouts/         # Layout strutturali
    └── pages/           # Rotte e logica di fetching build-time
```

## 🧞 Comandi Principali

| Comando | Azione |
| :--- | :--- |
| `npm run dev` | Avvia il server di sviluppo locale |
| `npm run build` | Genera la build statica in `./dist/` |
| `npm run preview` | Anteprima locale della build di produzione |
| `npm run astro check` | Validazione TypeScript e Astro |

## 🌟 Caratteristiche Implementate

- **Hero High Impact:** Visuale mozzafiato del tramonto di Sant'Elia.
- **Storytelling:** Narrativa del brand con animazioni reveal.
- **Product Highlights:** Griglia dei prodotti core (Cocktail, Food, Tradizione).
- **Sunset Experience:** Sezione esperienziale dedicata al momento clou del bar.
- **Gallery:** Visualizzazione immersiva della location.
- **Reviews:** Social proof da clienti soddisfatti.
- **Location & Info:** Mappa interattiva e info operative.
- **Mobile-First Conversion:** CTA WhatsApp persistenti per prenotazioni rapide.
